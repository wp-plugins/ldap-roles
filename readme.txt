A plugin to map LDAP groups to Wordpress roles. This plugin depends on and reuses parts of WPMU LDAP Plugin http://wpmuldap.tuxdocs.net/

== Description ==
This plugin allows to associate roles per blog based on the user inclusion in one of several LDAP groups.

It builds upong the WPMU LDAP plugin available at http://wpmuldap.tuxdocs.net/ .

== Installation ==

1. Unzip the ZIP file in the `/wp-content/mu-plugins/` directory
2. The blog Network administrator can then associate roles to LDAP groups in the MU administration interface
